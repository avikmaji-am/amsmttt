# Tic Tac Toe Game 

## Project Overview
Ye ek **Tic Tac Toe Game** hai jo **sirf ek HTML file** me banaya gaya hai.  
Is file ke andar hi **HTML, CSS, aur JavaScript** sab included hain.  
Game me **do modes** available hain:
- VS Computer  
- 2 Player  

Ye project beginners ke liye simple, interactive aur easy-to-use hai.

---

## Features
- All-in-one single HTML file  
- Two game modes: VS Computer & 2 Player  
- Automatic win & draw detection  
- Winning line highlight hoti hai  
- Reset button se game restart  
- Clean aur simple user interface  

---

## Game Options and Their Functions

Game start hote hi screen par **do main buttons** dikhai dete hain:

### 1. VS Computer
- Is mode me **player computer ke against** khelta hai  
- Player ka symbol **X** hota hai  
- Computer ka symbol **O** hota hai  
- Player ke move ke baad computer automatically move karta hai  
- Ye **single-player mode** hai  

**Best for:**  
Akele game khelne ke liye

---

### 2. 2 Player
- Is mode me **do players same device par** khelte hain  
- Player 1 = **X**  
- Player 2 = **O**  
- Dono players **turn by turn** move karte hain  
- Computer ka koi role nahi hota  

**Best for:**  
Friends ya family ke saath khelne ke liye

---

### 3. Reset Button
- Current game ko **restart** karta hai  
- Board clear ho jata hai  
- Naya match start hota hai  
- Selected mode (VS Computer ya 2 Player) ke saath game dobara chalu hota hai  

---

### 4. Game Status Message
- Batata hai **kis player ki turn hai**  
- Winner milne par **Winner message** show hota hai  
- Draw hone par **Draw message** display hota hai  

---

## How to Play
1. `tic-tac-toe.html` file ko kisi bhi browser me open karein  
2. Game mode select karein:
   - VS Computer  
   - 2 Player  
3. Khali cell par click karke apna symbol place karein  
4. Jo player **3 symbols ek line me** (horizontal, vertical, ya diagonal) bana leta hai, wahi winner hota hai  
5. Agar board full ho jaye aur koi winner na mile, to game **Draw** hota hai  
6. **Reset** button par click karke game dobara start karein
